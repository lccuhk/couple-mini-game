# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [Unreleased]

### Added
- GitHub Actions workflow for automatic GitHub Pages deployment
- Project preview SVG image
- GitHub badges (stars, forks, issues, license, last-commit)
- Tech stack badges (HTML5, CSS3, JavaScript, etc.)
- MIT LICENSE file

### Changed
- Enhanced .gitignore with more comprehensive rules
- Added project preview screenshot section to README

## [3.0.0] - 2026-06-20

### Added
- 20 new games (21-40): Dou Dizhu, Tetris, Tank Battle, Linkup, Basketball, Soccer, Bubble Shooter, Match 3, Sokoban, Breakout, Racing, Space Shooter, Bomberman, Hop Battle, Runner, Tower Defense, Gomoku, Chinese Chess, Go, Fishing
- Test report for 40 games
- Comprehensive test guide (TEST_GUIDE.md)
- Automated test tools (test_games.py, test_games.html)

### Changed
- Updated README with complete 40-game list
- Enhanced game selection interface with better categorization

## [2.0.0] - 2026-06-10

### Added
- 6 new games: Uno, Blackjack, Monopoly, Bowling, Badminton, Pictionary
- Beautiful card styling for Blackjack game
- Fixed game icon for Blackjack

### Changed
- Enhanced visual design for card games
- Improved game loading performance

## [1.5.0] - 2026-06-05

### Added
- 4 new games: 2048 Battle, Darts Battle, Puzzle Battle, Dice Battle
- Pause feature to Tic Tac Toe game
- Share feature to Tic Tac Toe game

### Fixed
- Duplicate togglePause function in Pong game
- Missing HTML elements in Pong game
- Theme features in Pong game

## [1.0.0] - 2026-05-20

### Added
- Initial release with 10 classic games:
  - Pong
  - Tic Tac Toe
  - Rock Paper Scissors
  - Snake Battle
  - Shooter Battle
  - Memory Match
  - Connect Four
  - Whack-a-Mole
  - Reaction Test
  - Number Bomb
- Game selection homepage with neon theme
- Responsive design for mobile and desktop
- Local multiplayer support
- Score tracking for each game
- Theme switching (light/dark/neon)
