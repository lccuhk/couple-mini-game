# 🎮 双人小游戏测试指南

## 概述
提供了两种测试工具来验证21-40号新游戏能否正常启动。

---

## 📋 文件说明

### 1. `test_games.py` - Python 命令行测试工具
功能完整的自动化测试脚本。

### 2. `test_games.html` - Web 界面测试工具
美观的交互式测试页面。

---

## 🚀 使用方法

### 方法一：Web 界面测试（推荐）⭐

1. 确保本地服务器正在运行（端口 8000）
2. 在浏览器中打开：
   ```
   http://localhost:8000/test_games.html
   ```
3. 点击测试按钮即可！

### 方法二：Python 命令行测试

#### 前置要求
- Python 3.x

#### 运行测试

```bash
cd /path/to/couple-mini-game
python3 test_games.py
```

#### 测试模式
1. **🔍 检查模式** - 检查所有20款游戏
2. **🎲 随机模式** - 随机选择10款测试
3. **👆 手动模式** - 逐个打开游戏在浏览器中测试
4. **📋 列出游戏** - 显示所有游戏列表

---

## 🎯 测试内容

每款游戏会检查：
- ✅ 文件是否存在
- ✅ 文件大小是否正常
- ✅ HTTP 请求是否成功
- ✅ 响应内容是否有效（包含 DOCTYPE 且大于 1KB）

---

## 📊 测试结果

### Python 脚本
- 测试结束后自动保存结果到 `test_results_YYYYMMDD_HHMMSS.txt`
- 控制台显示详细进度

### Web 界面
- 实时显示测试进度
- 彩色状态指示（紫色=待测试，粉色=测试中，绿色=通过，红色=失败）
- 统计面板显示通过/失败数量
- 详细结果面板

## 🔧 故障排除

### 问题：Python 脚本无法连接服务器
- 确保本地服务器正在运行
- 检查端口是否为 8000
- 确认项目路径正确

### 问题：Web 页面显示异常
- 刷新页面
- 检查浏览器控制台（F12）是否有错误
- 确认服务器正在运行

### 问题：游戏测试失败
- 检查 `games/` 目录下是否存在对应文件
- 验证文件是否完整
- 尝试直接在浏览器打开游戏页面
